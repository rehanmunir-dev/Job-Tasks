*   **Key Features**:

    *   Built with React Context API and Functional Hooks.

### 2. User Registration Form
A professional-grade registration form focusing on high-end UI/UX and robust validation.
*   **Live Link**: [Registration Form Demo](https://rehanmunir-dev.github.io/Job-Tasks/form)
*   **Key Features**:
    *   Real-time field validation (Email, Phone, Name, Password).
    *   Animated password strength indicator.
    *   Password visibility toggling.
    *   Interactive focus states and error shaking animations.
    *   Beautiful success overlay with animated checkmarks.
    *   Styled with **Tailwind CSS v4**.

## 💻 Local Development

To run any of the projects locally, follow these steps:

### Prerequisites
*   Node.js (v18 or higher)
*   npm

### Setup
1.  Clone the repository:
    ```bash
    git clone https://github.com/rehanmunir-dev/Job-Tasks.git
    cd Job-Tasks
    ```

2.  Navigate to a project folder (e.g., the form):
    ```bash
    cd User-Registration-Form
    ```

3.  Install dependencies:
    ```bash
    npm install
    ```

4.  Start the development server:
    ```bash
    npm run dev
    ```
