## 📂 Project Showcase

### 1. 📊 Product Explorer Dashboard
A sophisticated data listing dashboard that fetches real-time products from the `FakeStoreAPI`. It focuses on efficient data handling and a modern browsing experience.

*   **Live Demo**: [Explore Dashboard](https://rehanmunir-dev.github.io/Job-Tasks/dashboard)
*   **Key Technical Features**:
    *   **Context API Management**: Global state handling for products and filters.
    *   **Live API Integration**: Asynchronous data fetching with loading and error states.
    *   **Smart Search**: Dynamic filtering of products by name and category.
    *   **Sorting System**: Organize data by price (low/high) and user ratings.
    *   **Premium Glassmorphism**: Cards styled with frosted glass effects and subtle glow.
*   **Best Practices**: Debounced logic (planned), functional hooks, and semantic HTML5.

### 2. 📝 Advanced User Registration Flow
A high-performance form application that prioritizes user feedback and security through interactive validation.

*   **Live Demo**: [Try Registration Form](https://rehanmunir-dev.github.io/Job-Tasks/form)
*   **Key Technical Features**:
    *   **Real-time Validation**: Inline error messaging that responds instantly as users type or blur fields.
    *   **Password Engineering**: Visual strength meter with 5 levels of security assessment.
    *   **Micro-Animations**: Error-state shaking, smooth field transitions, and checkmark animations.
    *   **Visual Polish**: Custom-generated abstract background image and Tailwind CSS v4 styling.
    *   **Success Logic**: Simulated asynchronous submission with a professional success overlay.

## ⚙️ Local Development Instructions

To clone and run these projects locally:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/rehanmunir-dev/Job-Tasks.git
    cd Job-Tasks
    ```

2.  **Explore a Task**:
    ```bash
    # For the Dashboard
    cd Product-Explorer-Dashboard
    
    # OR For the Form
    cd User-Registration-Form
    ```

3.  **Install & Run**:
    ```bash
    npm install
    npm run dev
    ```
